# django-unoletutils

Django app with utils for Unolet.


## Requirements

Python 3.6 or newer with Django >= 2.2 or newer.


## Installation

1. Install using pip:

    ```bash
    pip install django-unoletutils
    ```
    
2. Add to `INSTALLED_APPS` in your `settings.py`:

   ```python
   INSTALLED_APPS = (
       # ...
       "unoletutils",
       # ...
   )
   ```
3. Use views:
   
   ```python
   from unoletutils.libs import utils
   ```
